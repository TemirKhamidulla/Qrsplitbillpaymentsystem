Design a modern restaurant admin dashboard for managing a QR-based split bill payment system.

Platform: Web desktop dashboard.

Context:
This system is used in restaurants in Kazakhstan. The admin panel allows restaurant managers to manage waiters, menu items, tables, and orders. The system is connected to a waiter tablet app and a guest QR payment page.

Main goal:
Allow restaurant administrators to easily manage staff and menu items.

Create the following screens:

1. Admin Login Page
Simple login with email and password.

2. Dashboard
Show key metrics:
- Today’s revenue
- Number of active tables
- Total orders today
- Average bill
Use clean analytics cards and a simple revenue chart.

3. Staff Management
List of waiters.
Columns:
- Name
- Status
- Phone
- Actions (Edit / Delete)

Include a button:
+ Add Waiter

Add Waiter Form:
- Name
- Email
- Phone
- Password
- Role (Waiter / Manager)

4. Menu Management
Table layout showing menu items.

Columns:
- Dish Name
- Category
- Price
- Availability
- Actions

Include button:
+ Add Menu Item

Add Item Form:
- Dish Name
- Category
- Price
- Description
- Photo
- Availability toggle

5. Categories Management
List of menu categories:
- Soups
- Salads
- Main dishes
- Drinks
- Desserts

Allow adding and editing categories.

6. Table Management
Grid showing restaurant tables (Table 1, Table 2, etc.).
Allow editing number of seats.

7. Orders History
Table showing:
- Order ID
- Table number
- Waiter
- Total amount
- Status
- Date

8. Payments
Transaction history:
- Payment ID
- Client
- Table
- Amount
- Payment method (Card / QR / Cash)
- Status

Design style:
Clean, modern SaaS dashboard.
Minimal design similar to Stripe or Shopify admin panels.
Use a left sidebar navigation and main content area.

Colors:
Neutral UI with green accent elements (inspired by digital payments like Kaspi).

Typography:
Clear, professional dashboard typography.

Make it look like a real professional restaurant management system.
